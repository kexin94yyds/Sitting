# Electron 桌面应用使用说明

这是一个基于 Electron 的久坐提醒桌面应用，支持 macOS 和 Windows。

## 安装依赖

```bash
npm install
```

## 开发运行

```bash
npm run electron
```

## 打包应用

### macOS
```bash
npm run build:mac
```

打包完成后，安装包位于 `dist/` 目录下：
- `久坐提醒-1.0.0.dmg` - macOS 安装包
- `久坐提醒-1.0.0-mac.zip` - macOS 压缩包

### Windows
```bash
npm run build:win
```

打包完成后，安装包位于 `dist/` 目录下：
- `久坐提醒 Setup 1.0.0.exe` - Windows 安装程序
- `久坐提醒-1.0.0-win.zip` - Windows 便携版

## 功能特性

1. **定时提醒**：设置提醒间隔（1-180分钟），到时间自动发送系统通知
2. **循环提醒**：启用后，每5分钟自动提醒一次
3. **系统托盘**：应用最小化到系统托盘，不占用任务栏
4. **后台运行**：关闭窗口后应用继续在后台运行，定时器正常工作
5. **设置持久化**：应用重启后自动恢复之前的设置和运行状态

## 使用说明

1. 启动应用后，设置提醒间隔（分钟）
2. 可选择是否启用循环提醒
3. 点击"开始计时"按钮启动提醒
4. 应用可以最小化到系统托盘，点击托盘图标可以显示/隐藏窗口
5. 右键托盘图标可以快速开始/暂停提醒

## 技术栈

- Electron 28+
- electron-store（配置存储）
- electron-builder（打包工具）

## 注意事项

- macOS 首次运行时，需要在"系统设置 > 通知"中允许应用发送通知
- Windows 首次运行时，需要在"设置 > 系统 > 通知"中允许应用发送通知
- 应用关闭窗口后仍在后台运行，需要右键托盘图标选择"退出"才能完全关闭
